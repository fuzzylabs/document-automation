# Usage

## Prerequisites

* `python`
* `virtualenv`
* `pip`

## Setup

```
virtualenv env
source env/bin/activate
pip install -r requirements.txt
```

## Running

### Environment variables

```
GOOGLE_APPLICATION_CREDENTIALS
GOOGLE_PROJECT_ID
GOOGLE_MODEL_ID
```

### Classify an image

```
python classify.py <input image>
```
